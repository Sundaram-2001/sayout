<h1 align="center" id="title">sayout website</h1>

<p id="description">This project is like a clone to the anonymous messages website sayout.net .</p>

<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository</p>

```
git clone https://github.com/Sundaram-2001/sayout.git
```

<p>2. Setup Frontend</p>

```
cd frontend
```

<p>3. Setup Frontend</p>

```
npm install
```

<p>4. Run the frontend dev server</p>

```
npm run dev
```

  
  
<h2>💻 Built with</h2>

Technologies used in the project:

*   Frontend:Sveltekit
*   Backend:Golang
*   Database:MongoDB
